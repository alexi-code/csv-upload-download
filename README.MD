## CSV upload/download service

The goal is to create an application that allows uploading and getting a csv file.
The sample csv file can be found in a project root.

### Tech stack:

- Java 11
- Spring: Boot, Data
- Hibernate (JDBC, H2 in-memory RDBMS)
- OpenAPI 3 + Swagger GUI

### Build & run

1. Build app: ```./gradlew clean build ```
2. Run app using CLI: ```./gradlew bootRun ```
